36. Jarak bandara dan stasiun kereta api adalah 45 km. Skala yang digunakan 1 : 900.000 Berapa cm jarak bandara dan stasiun kereta api pada peta? tolong dijawab dengan cepat karena dikumpulkan jam12


Jawab:

40500

Penjelasan dengan langkah-langkah:

Jawab:

Jarak bandara dan stasiun kereta api pada peta adalah 5 cm.

Penjelasan dengan langkah-langkah:

Diketahui : JS = 45 km dan S = 1 : 900.000.

Ditanya : JP = ...?

Jawab : JP = JS ÷ S = 45 km ÷ 900.000 = 4.500.000 ÷ 900.000 = 5 cm. Jadi, jarak bandara dan stasiun kereta api pada peta adalah 5 cm.
