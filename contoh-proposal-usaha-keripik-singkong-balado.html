26. Ibu Dina memproduksi dua jenis keripik singkong, yaitu rasa original dan balado. Setiap kilogram keripik rasa original membutuhkan modal Rp. 10.000 sedangkan keripik rasa balado membutuhkan modal Rp. 15.000 per kilogram. Modal yang dimiliki ibu Dina Rp. 500.000. Setiap hari hanya bisa memproduksi keripik singkong paling banyak 40 kilogram. Keuntungan tiap kilogram keripik singkong rasa original adalah Rp. 2.500 dan rasa balado adalah Rp. 3.500. Tentukan keuntungan terbesar yang diperoleh ibu Dina ​


Jawaban:

Pembahasan :

Misal

x = keripik pisang rasa coklat

y = keripik pisang rasa keju

Modal :

keripik rasa coklat = Rp10.000,00

keripik rasa keju = Rp15.000,00

Modal ibu yang dimiliki = Rp500.000,00

10.000x + 15.000y ≤ 500.000

10x + 15y ≤ 500

2x + 3y ≤ 100 ........... (1)

(≤ : daerah diarsir ke bawah)

x = 0

=> 2(0) + 3y = 100

=> y = 100/3

=> (0, 100/3)

y = 0

=> 2x + 3(0) = 100

=> x = 50

=> (50, 0)

Tiap hari dapat memproduksi paling banyak 40 kg

x + y ≤ 40 ................ (2)

(≤ : daerah diarsir ke bawah)

x = 0

=> 0 + y = 40

=> y = 40

=> (0, 40)

y = 0

=> x + 0 = 40

=> x = 40

=> (40, 0)

x ≥ 0, y ≥ 0 .............. (3)

Fungsi Sasaran => Keuntungan

Keripik rasa coklat = Rp2.500,00

Keripik rasa keju = Rp3.000,00

f(x, y) = 2.500x + 3.000y

Ditanyakan :

Keuntungan terbesar = ... ?

Titik potong garis (1) dan (2)

2x + 3y = 100 |×1|

x + y = 40 |×2|

2x + 3y = 100

2x + 2y = 80

-------------------- -

....... y = 20

x + y = 40

x + 20 = 40

x = 20

(20, 20)

Berdasarkan gambar (lihat pada lampiran)

(0, 100/3), (20, 20) dan (40, 0)

Kita substitusikan ke fungsi sasaran

f(x, y) = 2.500x + 3.000y

f(0, 100/3) = 2.500(0) + 3.000(100/3) = 100.000

f(20, 20) = 2.500(20) + 3.000(20) = 110.000

f(40, 0) = 2.500(40) + 3.000(0) = 100.000

Jadi keuntungan maksimumnya adalah Rp110.000,00

33. Kegiatan ekonomi dari usaha keripik singkong


Jawaban:

Usaha keripik singkong dapat mencakup berbagai kegiatan ekonomi, antara lain:

1. Produksi: Kegiatan ini melibatkan pengolahan singkong menjadi keripik. Singkong dikupas, dipotong tipis, dan kemudian digoreng atau dikeringkan untuk menghasilkan keripik singkong yang siap dikemas.

2. Pembelian bahan baku: Untuk memproduksi keripik singkong, pemilik usaha perlu membeli singkong segar sebagai bahan baku. Ini dapat melibatkan kerjasama dengan petani lokal atau penyedia bahan baku singkong.

3. Pemasaran dan penjualan: Setelah keripik singkong diproduksi, langkah selanjutnya adalah memasarkan dan menjual produk tersebut. Pemilik usaha dapat menjual keripik singkong secara langsung di toko atau warung mereka, melalui distributor, atau bahkan melalui platform e-commerce.

4. Manajemen operasional: Usaha keripik singkong juga melibatkan kegiatan manajemen operasional seperti pengaturan produksi, pengawasan kualitas, manajemen persediaan bahan baku dan kemasan, serta manajemen keuangan.

5. Penyediaan lapangan kerja: Jika usaha keripik singkong berkembang, pemilik usaha mungkin perlu mempekerjakan karyawan tambahan untuk membantu dalam produksi, pengemasan, atau kegiatan lainnya. Ini dapat memberikan manfaat ekonomi dalam bentuk menciptakan lapangan kerja dan memberikan penghasilan kepada karyawan.

6. Riset dan pengembangan: Untuk menjaga daya saing dan menciptakan inovasi dalam produk keripik singkong, pemilik usaha mungkin terlibat dalam kegiatan riset dan pengembangan. Hal ini meliputi eksperimen dengan resep baru, mencari cara baru dalam pengolahan atau pengemasan, atau mengeksplorasi pasar baru.

Dalam keseluruhan, usaha keripik singkong dapat memberikan kontribusi ekonomi yang signifikan dengan melibatkan berbagai kegiatan produksi, pemasaran, manajemen, dan pengembangan usaha.
