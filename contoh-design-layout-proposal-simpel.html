2. Langkah mendesign layout​


Jawaban:

1. Pilih menu File ► New.

2. Dalam kotak New Document yang tampil, buat setting halaman sebagai berikut:

3. Ubah warna kanvas menggunakan warna #000033, Klik palet warna Canvas di dalam panel Properties, dan pilih warna yang telah ditentukan.

4. Tambahkan sebuah objek Rectangle menggunakan Rectangle tool dalam panel Tools grup Vector. Buat ukuran objek menjadi W=874 dan H=663, dengan posisi X=13 dan Y=279.

5. Tambahkan kembali sebuah objek Rounded Rectangle menggunakan Rounded Rectangle tool dalam panel Tools grup Vector. Buat ukuran objek menjadi W=874 dan H=321, dengan posisi X=13 dan Y=52.

6. Atur properti kedua objek di atas dengan menggunakan panel Properties. Pasang ukuran stroke menjadi 2, dengan warna stroke #666666.

7. Pilih kedua objek tersebut menggunakan kombinasi Shift dan klik pada keyboard, kemudian pilih menu Modify ►Combine Path ►Union.

Hasilnya adalah kedua objek tersebut akan disatukan hingga membentuk sebuah objek baru.

8. Imporlah sebuah gambar ke dalam kanvas dengan menggunakan menu File ►Import. Dalam kotak dialog Import, pilih gambar kemudian klik tombol Open.

9. Klik tahan geser mouse untuk meletakkan objek gambar tersebut di dalam kanvas. Buat ukuran objek menjadi W=874 dan H=668, dengan posisi X=13 dan Y=0.

Jika posisi gambar belum sesuai dengan kebutuhan dalam grafis halaman, geser kembali untuk mendapatkan hasil yang lebih bagus.

10. Pilih objek gambar, lakukan pemotongan gambar untuk menyesuaikan ukuran kebutuhannya dalam desain. Pilih menu Edit ► Crop Selected Bitmap, sehingga gambar akan dikelilingi handel untuk memilih area hasil pemotongan.

11. Ubah area pemotongan sesuai kebutuhan dalam desain Anda, sebagai contoh buat objek menjadi W=874 x H=200, dengan posisi X=13 dan Y=40; Tekan Enter pada keyboard untuk menampilkan hasil pemotongan.

12. Lakukan masking antara objek gambar dan objek vektor hasil langkah ke-7, Pilih objek gambar dan pilih menu Edit ► Cut, selanjutnya pilih objek vektor dan pilih menu Edit ► Paste Inside.

Hasil masking tersebut akan d’igunakan sebagai header dalam grafis halaman web Anda.

13. Pilih menu File ► Save As untuk menyimpan desain Anda dengan nama index.png.

15. Sebutkan & Jelaskan fitur-fitur Design & Page Layout​


Fitur-fitur yang terdapat pada tab Design di Microsoft Word, antara lain:

•Themes : digunakan untuk mengatur tema halaman agar menjadi lebih menarik.

•Document Formatting : untuk mengatur format pada dokumen.

•Color : digunakan untuk memberi warna pada format dokumen yang sudah dibuat.

•Font : mengatur font atau gaya tulisan pada halaman.

•Effect : untuk memberikan efek pada halaman

•Paragraph Spacing : mengatur spasi(jarak) antar paragraf sesuai yang kita inginkan.

•Watermark : menambahkan tulisan semar-semar di tengah-tengah halaman.

•Page Color : memberi warna pada halaman.

•Page Border : menambahkan bingkai pada pinggir halaman.

Fitur-fitur yang terdapat pada tab Page Layout di Microsoft Word, antara lain:

•Margin : fitur ini digunakan untuk mengatur batas-batas pada pinggir halaman. Batas-batasnya meliputi batas atas, batas bawah, batas kiri, dan batas kanan.

•Orientation : fitur ini mengatur orientasi bagaimana orientasi pada kertas, apakah kertas miring ataupun tegak hal tersebut yang dapat dilakukan dari fitur ini.

•Size : fitur untuk mengatur ukuran kertas. Ada jufa ukuran standar yang sudah disediakan dan juga dapat mengatur ukurannya sendiri dengan bebas.

•Column : membuat halaman menjadi memiliki kolom.

•Watermark : untuk memberikan tulisan kabur(tanda air) pada belakang dokumen.

•Page color : digunakan untuk memberikan warna pada halaman.

•Page border : untuk mengatur dan memberikan garis pada pinggir halaman atau juga dapat disebut bingkai.

•Ident Left : digunakan untuk mengatur batas pada kiri tulisan.

•Ident Right : digunakan untuk mengatur batas pada kanan tulisan.

•Spacing Before : untuk mengatur jarak antara paragraf dengan paragraf sebelumnya.

•Spacing After : untuk mengatur jarak antara paragraf dengan paragraf sesudahnya.

18. komponen design, layout, referensi, meilings, review dan view itu apa saja​


Jawaban:

Elemen layout dalam desain grafis dapat berisi sejumlah elemen yang berbeda. Beberapa elemen desain layout adalah sebagai berikut:

Teks (text): Elemen teks termasuk judul, heading, dan paragraf. Dalam desain text-heavy, elemen-elemen ini harus ditata dengan cara intuitif yang mudah dibaca.

Gambar (image): Foto dan visual lainnya, seperti ilustrasi, membantu memecah teks dan mengkomunikasikan pesan tertentu. Elemen ini sangat efektif dalam melibatkan pemirsa melalui emosi.

Garis (line): Garis digunakan untuk membagi bagian. Mereka juga dapat menambahkan penekanan pada teks.

Bentuk (shape): Bila digunakan dengan baik, bentuk menambahkan sentuhan bakat dan kepribadian pada tata letak. Persegi panjang dan lingkaran adalah yang paling umum, tetapi ada banyak cara untuk berkreasi dengan bentuk.

Ruang putih (white space): Seperti yang telah kita bahas, menambahkan ruang di antara bagian-bagian tata letak mencegah pengguna kewalahan atau lelah saat melihat desain yang sibuk.

Format dan konteks desain menentukan bagaimana elemen-elemen ini diimplementasikan dalam layout. Saat memutuskan bagaimana menyeimbangkan berbagai elemen desain, penting untuk selalu mempertimbangkan konteks dan pengalaman pengguna

27. apa perbedaan antara struktural design dan decorative design sebutkan pula contohnya.​


Jawaban:

Jawaban

Structural Design (desain struktur) adalah susunan dari garis, bentuk, warna, dan tekstur

dari suatu benda baik berupa benda yang mempunyai ruang maupun gambaran dari suatu
