3. apakah perbedaan cover page blank page & page break​

Fungsi Cover Page, Blank Page & Page Break di Ms. Word

Berkaitan dengan Cover; kali ini Office Praktis bakal jelasin apa yang dimaksud dengan Cover Page, Blank Page, dan Page Break serta fungsinya masing2.

Pengertian Cover Page, Blank Page, dan Page Break serta Fungsinya

Cover Page adalah suatu cara untuk membuat halaman "sampul depan" dengan template yang sudah disajikan secara default (bawaan) oleh program aplikasi Microsoft Word.

Nah ada lagi yang dinamakan Blank Page. Blank Page adalah suatu cara untuk menampilkan dan/atau membuat halaman terbaru pada dokumen yang sama.

Tindakan dari Blank Page pun tidak akan mengganggu halaman lainnya; dan bahkan, penomoran dari tiap2 halaman pada dokumen tersebut akan berubah secara otomatis.

Mau bukti dari Blank Page? Coba ketikan sesuatu pada halaman pertama di Ms. Word. Lalu tekan Blank Page, secara otomatis kalian akan diarahkan ke halaman selanjutnya tanpa peduli ruang spasi pada halaman pertama tadi.

Bisa dibilang, dengan Blank Page; kita gak perlu lagi tekan tombol enter-enter dan enter untuk menuju ke halaman selanjutnya.

Terakhir ada Page Break. Tahu gak sih apa itu Page Break?

Pengertian dan fungsi dari Page Break itu hampir mirip dengan Blank Page; hanya saja, Blank Page bisa dilakukan secara bebas, sedangan Page Break harus mengasarkan kursor pada paragraf tertentu untuk membuat halaman baru selanjutnya.

Page Break bisa dibilang membuat halaman baru. Caranya? Buat 5 baris paragraf. Pada paragraf ketiga, arahkan kursor dibagian terakhir, dan klik tombol Page Break.

Maka baris paragraf pertama dan kedua akan dihalaman pertama, sedangkan baris paragraf ketiga, keempat, dan kelima akan diarahkan ke halaman selanjutnya (pada dokumen yang sama).

9. langkah langkah membuat cover page sederhana


Jawaban:

1.Pada Insert tab, Pages group, klik Cover Page untuk menampilkan galeri template cover page.

2. Klik salah satu template yang diinginkan. Cover page akan otomatis ditempatkan di halaman pertama dokumen.

3.Jika ingin menempatkannya di halaman lain: pada galeri klik kanan thumbnail cover page dan klik pilihan yang diinginkan.

4.isi informasi pada field di cover page seperti judul, sub judul, dan lain-lain. Untuk membuat tanggal, klik tanda panah dan pilih tanggal pada kalender.

5.Beri gaya format yang diinginkan (bold, italic, underline) atau ganti warna dan ukuran font.

6. Beberapa template cover page menyediakan gambar, bila ingin menggantinya, klik kanan pada gambar dan pilih Change Picture.

7. Bila tidak tersedia gambar, kita dapat menambahkan gambar dengan cara: pada Insert tab, Illustration group, klik Insert Picture.

8. Klik tombol Save untuk menyimpan cover page.

9. Jika ingin menyimpannya ke galeri agar dapat digunakan lagi, ikuti cara berikut:

Pilih seluruh isi cover page.

Pada Insert tab, klik Cover Page dan pilih Save Selection to Cover Page Gallery.

Pada kotak dialog Create New Building Block, beri nama cover page di kotak Name dan klik OK.

10. Untuk menghapus cover page: Pada Insert tab, klik Cover Page dan pilih Remove Current Cover Page.

Cara pembuatan :

2.      Masukan telur dan isi air es secukupnya sambil diaduk, masukan margarine sampai elastic diamkan 15 menit.

3.      Kempiskan adonan. Membentuk bulat adonan seberat 15 grm, lalu diamkan selama 10 menit.

F.      MENENTUKAN HARGA POKOK DAN HARGA JUAL

Modal awal = Rp. 100.000,- menghasilakan sebanyak 70 buah donat.

Perhitungan :

Menentukan harga pokok   = modal awal : banyaknya produk yang dihasilkan

= Rp. 100.000,- : 70

= Rp. 1.428,57 di bulatkan menjadi Rp. 1.500,-

Menentukan harga jual       = harga pokok + (harga pokok x 33%)

                                           = Rp. 1.500,- + (Rp. 1.500,- x 33%)

                                           = Rp. 1.500,- + Rp. 495,-

                                           = Rp. 1.995,- di bulatkan menjadi Rp. 2.000,-

kesimpulan :

Harga pokok           = Rp. 1.500,-

Harga jual               = Rp. 2.000,-

29. hal yang perlu dituliskan dalam cover sheet proposal adalah​


1. Judul Proposal

Judul proposal ditulis singkat, jelas, dan padat.

2. Latar Belakang

Pada bagian ini penulis proposal menjelaskan latar belakang diadakannya kegiatan tersebut.

3.Dasar Pemikiran

Dalam bagian ini dicantumkan dasar hukum diadakannya kegiatan tersebut.

4.Jenis Kegiatan

Pada bagian ini dijelaskan berbagai jenis kegiatan beserta keterangan lainnya.

5. Tujuan Kegiatan
